Open v11
What changed
  • Custom brand font (The Care Ranch) applied to all headings, kickers, drop caps, method numerals, and team initials via @font-face. Body copy and captions are now Arial. Google Fonts link removed.
  • GSAP + ScrollTrigger + Lenis added via CDN for scroll-triggered animations: section headers fade up, method/team/benefit/journey/info/statement grids stagger their children, hero content fades up on load, hero background has a subtle parallax on scroll, Lenis provides smooth-scroll.
  • Respects prefers-reduced-motion — visitors with that OS setting see everything immediately with no motion.
  • Download PDF button fixed in the bottom-right. Triggers window.print() with a dedicated print stylesheet that strips the hero background, stacks every grid to one column, forces page breaks at sections, preserves brand colors, and caps image heights so no single image dominates a page.
  • Responsive audit and fixes: hero h1 no longer overflows at <400px widths (letter-spacing and font size reduced), team card tablet layout uses minmax(160px, 200px) for the portrait column so it survives narrow tablets, drop cap scales down on mobile, journey step markers shrink on mobile, CTA button sizes down on mobile.
